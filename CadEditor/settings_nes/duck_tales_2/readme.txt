Dt2.py - python-модуль для декомпрессии видеопамяти Duck Tales 2.
 
Для того, чтобы извлечь банки видеопамяти из рома, надо положить ром в папку с модулем, назвать файл "Duck Tales 2.nes" и запустить на выполнение скрипт dt2.py. Файл с разсжатыми банками памяти будет называться VideoBack_DT2.bin. 
Конфиг редактора Settings_DuckTales2-All.cs загружает данные видеопамяти из этого файла.

Для сжатия видеопамяти можно воспользоваться функцией compress из модуля dt2.py, после этого сжатые данные нужно вставить вместо старых в ром (хекс-редактором).
