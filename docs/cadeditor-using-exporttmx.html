Экспорт карт в формат TMX

Видеопример экпорта карт игры Duck Tales 2 в формат TMX

CadEditor включает в себя плагин для экспорта карт в формат TMX, используемых в редакторе Tiled.

Плагин использует модуль IronPython для загрузки и выполнения кода на языка Python и библиотеку pytmxlib для генерации tmx-файла. Плагин использует параметры отображения блоков при экспорте (отображение сетки на блоках, масштаб картинок, используемые банк видеопамяти) из главного окна редактора - в нём можно настроить вид генерируемой карты.

Плагин использует описание раскладок уровней (Layout - форма уровня, составленная из разных экранов), описанные в конфиге игры. Если в конфиге игре не указана раскладка, испольуется раскладка по умолчанию, содержащая один первый игровой экран).

Обратный импорт карты из Tiled в редактор CadEditor невозможен, так как Tiled не поддерживает описание задание каких-либо огранений карты, необходимых для сохранения карт в ROM-файлы.

Пример экспорта всех карт из игры Duck Tales 2 и последующего окрытия их в редакторе Tiled показан на видео.
